Cara Instalasi SEB (Safe Exam Browser):

1. Gunakan Komputer atau Laptop dengan Sistem Operasi minimal Windows 7 dengan update dan Memory minimal 4 GB. Windows 10 sangat disarankan
2. Pastikan fungsi audio dan webcam telah tersedia dan beroperasi baik pada perangkat.
3. Download file sebsetup.exe dan sebpensisru21.seb (jadikan dalam satu folder di laptop atau komputer yang mudah diingat).
4. Install sebsetup.exe di laptop atau komputer anda. Ikuti seluruh petunjuk instalasi dengan seksama.
5. Untuk mengikuti ujian CBT Taruna Nusantara, Jalankan Safe Exam Browser dengan mengklik sebpensisru21.seb ! (ada dalam satu folder dengan file file sebsetup.exe.exe)
6. Ujian CBT Taruna Nusantara hanya bisa dilakukan dengan menggunakan Safe Exam Browser dan file seb yang disediakna oleh panitia Pensisru SMA TN
